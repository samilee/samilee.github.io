Surgent CPA Review Dashboard

Wireframes   |   UI Design

Overview

After a full website redesign and re-branding, Surgent needed an update to their CPA Review Dashboard that would match the new branding. This iteration was meant to serve as a quick update to be available to an upcoming wave of users. I was able to re-work their current dashboard to create a more user friendly experience and add additional functionality to be implemented later on. Beyond applying the new brand to their review application, I re-worked the interface to enhance and simplify the users experience.
